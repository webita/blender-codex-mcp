# Blender Codex MCP - Terms Of Use And Privacy

Maintained by Webita - Giorgio Sanna.

Blender Codex MCP is a local MCP integration for connecting Codex to Blender. It is provided as open-source software, without warranty.

## Privacy

This Codex-focused fork is configured with telemetry disabled by default. The recommended Codex configuration also sets `DISABLE_TELEMETRY=true`.

If you explicitly enable telemetry through environment variables or code changes, you are responsible for understanding what data is sent and where it is sent.

## Safety

Blender Codex MCP can execute Python code inside Blender through the `execute_blender_code` tool. Save important `.blend` files before running large or destructive operations.
